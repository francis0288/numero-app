# NumeroApp — Professional numerology reading tool EN / 中文 / Tiếng Việt

A full-stack numerology application supporting English, Simplified Chinese, and Vietnamese.

## Structure

```
numero-app/
├── apps/
│   ├── web/          # Next.js 14 web app
│   └── mobile/       # Expo mobile app
└── packages/
    ├── core/         # Core numerology logic
    └── i18n/         # Internationalization
```

## Getting Started

```bash
pnpm install
pnpm dev
```
